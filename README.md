##### SeekBarView
    
    1、刻度可变，递增
    
    2、双向滑动，即前后均可横向滑动
    
    3、滑动过程显示当前进度范围
    
    4、可设置初始值，即最小最大刻度
    
    5、可设置刻度组合，即每两个刻度之间距离
    
    6、可限制滑动范围，即屏幕左右的边界和左右指示针之间的间距
    
    7、可设置左右指示针中间最小间距，当滑动到此间距即不可滑动
    
    8、左右指示针相遇后是否可滑动，两者一起滑动
